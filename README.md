# RankForge

AI-powered SEO content platform that helps businesses rank higher on Google.

## What is RankForge?

RankForge generates SEO-optimized articles using AI, analyzes content for ranking factors, and provides keyword research — all in one platform. It turns a single keyword into a fully optimized, publish-ready article in minutes.

## Quick Start

```bash
cd rankforge
npm install
cp .env.example .env.local
# Fill in your environment variables
npx prisma db push
npm run dev
```

See `rankforge/` directory for the full application.
